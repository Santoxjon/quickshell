# FileViewAdapter

**Inherits:** [QtObject](https://doc.qt.io/qt-6/qobject.html)

Base type for adapters that can be used with [FileView.adapter](../FileView/FileView.md#properties).

## Signals

- **adapterUpdated**()
  
  This signal is fired when data in the adapter changes, and triggers [FileView.adapterUpdated()](../FileView/FileView.md#signals).
