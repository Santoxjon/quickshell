# Quickshell.Services.Pipewire - PwLinkGroup
**Version:** v0.3.0
**Description:** A group of connections between pipewire nodes.
## Properties
* `state` (PwLinkState): The current state of the link group.
* `target` (PwNode): The node that is *receiving* information. (the sink)
* `source` (PwNode): The node that is *sending* information. (the source)
