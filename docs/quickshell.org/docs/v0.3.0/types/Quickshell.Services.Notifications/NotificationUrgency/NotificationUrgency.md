# NotificationUrgency

**Version:** v0.3.0

**Description:**
The urgency level of a Notification.

This is a hint to the notification server about how important the notification is. For more details, see the freedesktop specification for this hint.

This is exposed on a notification object as Notification.urgency.

## Functions
* `toString(value)`: string

## Variants
* `Normal`
* `Critical`
* `Low`
