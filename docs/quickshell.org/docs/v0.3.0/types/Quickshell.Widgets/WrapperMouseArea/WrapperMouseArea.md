# WrapperMouseArea
**Version:** v0.3.0
**Description:** A MouseArea that wraps a single child item.

This component is used to wrap a single child item and apply margins to it.

## Properties
- `implicitHeight`: `real` - Overrides the implicit height of the wrapper.
- `bottomMargin`: `real` - The requested bottom margin of the content item.
- `child`: `Item` (readonly) - The child item.
- `implicitWidth`: `real` - Overrides the implicit width of the wrapper.
- `resizeChild`: `bool` - Determines if the child item should be resized larger than its implicit size.
- `extraMargin`: `real` - An extra margin applied in addition to the other margins.
- `topMargin`: `real` - The requested top margin of the content item.
- `leftMargin`: `real` - The requested left margin of the content item.
- `rightMargin`: `real` - The requested right margin of the content item.
- `margin`: `real` - The default for the margins.
