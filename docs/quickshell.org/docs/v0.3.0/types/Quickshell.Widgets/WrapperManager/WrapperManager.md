# WrapperManager
**Version:** v0.3.0
**Description:** A manager for a single child item.

This manager is used to implement components that have a single child item. It is not intended to be used directly.

## Properties
- `implicitHeight`: `real` - Overrides the implicit height of the wrapper.
- `child`: `Item` (readonly) - The child item.
- `implicitWidth`: `real` - Overrides the implicit width of the wrapper.
- `resizeChild`: `bool` - Determines if the child item should be resized larger than its implicit size.
