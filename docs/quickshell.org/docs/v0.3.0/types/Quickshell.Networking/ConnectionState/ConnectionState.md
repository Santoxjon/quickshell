# ConnectionState
The connection state of a device or network.

## Functions
| Type | Name | Description |
| --- | --- | --- |
| string | toString(state) | Converts a `ConnectionState` to a string. |

## Variants
| Name | Description |
| --- | --- |
| Unknown | The connection state is unknown. |
| Connecting | The device is connecting. |
| Disconnecting | The device is disconnecting. |
| Connected | The device is connected. |
| Disconnected | The device is disconnected. |
| Failed | The device failed to connect. |
