# WifiDeviceMode

The 802.11 mode of a `WifiDevice`.

## Functions

-   **toString**: `string` toString(mode)

## Variants

-   **AdHoc**
    The device is part of an Ad-Hoc network without a central access point.
-   **Station**
    The device is a station that can connect to networks.
-   **Unknown**
    The device mode is not known.
-   **AccessPoint**
    The device is an access point that other devices can connect to.
