# Quickshell.Services.SystemTray - Category
**Version:** v0.3.0
**Description:** Category of a StatusNotifierItem.
## Variants
* `Communications`: Communication related applications such as instant messengers or email clients.
* `Hardware`: Hardware controls like battery indicators or volume control.
* `ApplicationStatus`: Any other application, usually not long-running, that wants to display status information and does not fit into a different category.
* `SystemServices`: System services such as IMEs or disk indexing.
