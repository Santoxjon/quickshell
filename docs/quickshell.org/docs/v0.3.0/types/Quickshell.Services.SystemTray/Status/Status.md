# Quickshell.Services.SystemTray - Status
**Version:** v0.3.0
**Description:** Status of a StatusNotifierItem.
## Variants
* `Passive`: A passive item does not convey important information and can be considered idle. You may want to hide these.
* `Active`: An active item may have information more important than a passive one and you probably do not want to hide it.
* `NeedsAttention`: The item requires attention, for example because of an error condition.
