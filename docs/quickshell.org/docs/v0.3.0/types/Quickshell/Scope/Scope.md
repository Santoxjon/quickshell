# Scope

**Inherits:** [QObject](https://doc.qt.io/qt-6/qml-qtqml-qtobject.html)

Scope that propagates reloads to child items in order.

This can be used to set a common `reloadableId` for all children.

Note that this does not work for visible `Item`s (all widgets).
