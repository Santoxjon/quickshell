# Quickshell.Services.UPower - UPower
**Version:** v0.3.0
**Description:** Provides access to the UPower service.
## Properties
* `displayDevice`: `UPowerDevice` - UPower’s DisplayDevice for your system.
* `onBattery`: `bool` - If the system is currently running on battery power, or discharging.
* `devices`: `ObjectModel<UPowerDevice>` - All connected UPower devices.
