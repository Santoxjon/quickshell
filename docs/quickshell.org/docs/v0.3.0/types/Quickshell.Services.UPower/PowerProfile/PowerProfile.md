# Quickshell.Services.UPower - PowerProfile
**Version:** v0.3.0
**Description:** Power profile exposed by the PowerProfiles service.
## Functions
* `toString(profile)`: `string`
## Variants
* `PowerSaver` - This profile will limit system performance in order to save power.
* `Balanced` - This profile is the default, and will attempt to strike a balance between performance and power consumption.
* `Performance` - This profile will maximize performance at the cost of power consumption.
